# Concrete-Construction-Joint-Improvement-Solution
Цутгамал барилгын цутгалт хоорондын залгааг сайжруулах шийдэл

Энэхүү шийдэл нь бетон цутгамал барилгын цутгалт хоорондын(ихэвчлэн давхар хооронд) залгааг сайжруулах, бэхжүүлэх шийдэл юм. Уламжлалт арга нь хоёр цутгалтын босоо арматурыг илүү гарган цутгаж холбодог. Тиймээс цутгалтууд хоорондоо арматур төмрөөр бэхлэгддэг боловч бетоны хувьд үргэлжилсэн байдлаар цутгагдаагүй тул хоёр тусдаа бие болон хатдаг ба зарим тохиолдолд энэ нь сул тал болдог. 

Энэхүү асуудлыг уламжлалт углуургын арга ашиглан хоёр тусдаа хийгдсэн цутгалтыг бэхжүүлнэ. 

Цутгалт дууссаны дараа бетоныг хатахаас өмнө ханын зузаанаас хамааран төрөл бүрийн хэлбэр, хэмжээтэй углуургын суурийг шигтгэн суулгах ба уг суурь нь бетоны оройд углуургын хонхор (эм залгаа ч гэдэг) үүсгэх ба дараагийн цутгалт уг зайг дүүргэн цутгалт хоорондын холбоос болж өгнө.
